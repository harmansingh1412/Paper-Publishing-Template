# Turtle Example

This repository contains the codebase for a <strong>Turtle Graphics</strong> example, implemented using the TurtleGraphics package in R Studio. This package can be used to create sophisticated graphics by drawing lines by describing the location and orientation or the <strong>"turtle"</strong> in code. The graphic can be further coustomized by changing the line type and colors.
